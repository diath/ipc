ipc
====

Tibia IP changer for Linux available as a command line tool and a Qt-based GUI.

Building (command line)
========

* ``cd console/``
* ``g++ -std=c++14 -o../build/ipc ipc.cpp``

Building (GUI)
========

* ``cd gui/``
* ``qmake``
* ``make``
